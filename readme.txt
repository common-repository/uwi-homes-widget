UWI home loan application widget.

== Description ==

Integrates UWI home loan application to a wordpress home developer website.

PS: You'll be prompted to get an Uwi.com.ph API key to use it, once activated. Keys are available for Home Developers registered in UWI.

== Installation ==

Upload the Uwi Homes Widget plugin to your blog, activate it, and then enter your uwi.com.ph API key.

1, 2, 3: You're done!
